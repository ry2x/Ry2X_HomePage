# My Personal Homepage

This is my personal homepage project - a clean, modern website built to showcase my work and share my thoughts through blogging.

## Tech Stack

The site is built using modern web technologies:

- **Astro** - Static site generator with excellent performance
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript development
- **Markdown/MDX** - Content authoring

## Features

- Responsive design that works on all devices
- Fast loading times with optimized static generation
- SEO-friendly with proper meta tags and structured data
- Type-safe content management
- Clean, accessible UI design

## Architecture

The project follows best practices for maintainability:

- Component-based architecture
- Centralized content management
- Strong typing throughout
- Consistent styling system
- Well-documented code

## Getting Started

To run this project locally:

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

This project serves as both a personal website and a demonstration of modern web development practices.
