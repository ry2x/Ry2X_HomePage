# はじめまして

このたび、個人サイト **Ry2X.NET** を開設しました。

## 自己紹介

**Ry2X**（らいつーくろす）と申します。もっぱら **りず** と呼ばれています。

仕事では Java と TypeScript を使って Web フロントエンドからバックエンドまで幅広く開発しています。
趣味ではもっぱら **Rust** を触っていて、自分が欲しいと思ったツールを作るのが好きです。

開発以外では旅行と音ゲーが趣味です。旅先でいろんなものを食べたり、音ゲーで新しい曲を解禁したりするのが小さな楽しみになっています。あと、人に何かを教えるのも好きで、わかりやすく伝えるにはどうすれば良いかを考えること自体が楽しかったりします。

## このサイトについて

大きく 2 つのコーナーを用意しています。

### Projects

自分が作ったプロジェクトを紹介するページです。
ただ「こんなものを作りました」で終わらず、**設計の意図や技術的にこだわったところ**を掘り下げて書いていくつもりです。
Rust で作った CLI ツールや TUI アプリなど、少しずつ増やしていく予定です。

### Blog

気になった技術の話や、旅行の記録を書いていく場所です。
技術記事は「自分が調べたことをまとめる」くらいの温度感で、旅行記は「行ったところの記録」くらいの気軽さで続けていきたいと思っています。
